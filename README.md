# atlas

Personal home server for storage and backup.

## Requirements

- Debian 13.2

## Packages

- Avahi: facilitates service discovery on a local network via the mDNS/DNS-SD protocol suite
- ufw: program for managing a Netfilter firewall
- fail2ban: intrusion prevention
  - once exposing to internet, will be interesting to see bots attempting to login
